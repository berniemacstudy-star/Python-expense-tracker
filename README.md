# Personal Expense Tracker (Python)

A simple Python command-line program that helps users track expenses, calculate financial summaries, and export the results into JSON format.

## Features
- Enter multiple expenses
- Calculates total spending
- Displays percentage breakdown of expenses
- Provides basic financial advice
- Accepts multiple income sources
- Calculates savings
- Exports data into a JSON report

## Technologies Used
Python
JSON
File Handling
